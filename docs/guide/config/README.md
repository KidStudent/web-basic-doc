# 项目配置

## 本地配置

在项目的 public 文件夹中的 local-config.json 文件，此文件为项目本地的全局配置

### 配置解读
```json
{
  "Version": "1.2.0", //项目版本 
  "ShowWaterMark": false, //是否显示水印
  "ShowTabs": true, //是否显示标签页
  "ShowTheme": true //是否显示theme切换
}
```
