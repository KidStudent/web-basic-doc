# 开始

## 开发环境

与 [前端框架](/guide/start/#开发环境) 保持一致

## 本地开发

小智 `SDK` 是使用 `vite` 构建的 `vue3` 应用，最后是以 SDK 的形式展示的，所以需要引入的是打包之后的 `js` 文件，本地运行也是如此

安装依赖

```shell
npm install
```

打包生成 `SDK`

```shell
npm run build
```

启动项目

```shell
npm run dev
```
