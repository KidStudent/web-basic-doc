# 安装

::: warning 前提条件
由于本组件库是基于 [view-ui-plus](https://www.iviewui.com/view-ui-plus/guide/install) 和 [element-plus](https://element-plus.gitee.io/zh-CN/guide/installation.html) 开发的，所以要先引入它们
:::
